Quantitative Finance Projects

LSTM Model for Stock Price Prediction

This C++-based Long Short-Term Memory (LSTM) neural network predicts Amazon stock prices, achieving a 5\% RMSE improvement over baseline models. Data was fetched via curl and parsed with nlohmann/json, with statistical distributions from Boost. The model was trained on historical data, optimised for memory efficiency, and supports algorithmic trading applications. See code at \href{https://github.com/rhesus1/Finance/tree/main/LSTM}{github.com/rhesus1/Finance/LSTM}.

Option Pricing Models

These C++ models compute call and put option prices for Amazon using Black-Scholes (Analytical, Finite Difference, Monte Carlo) and Heston (Fourier, Monte Carlo, Finite Difference) methods, optimised with SIMD and OpenMP for a 30\% speedup. Visualisations include price surfaces and implied volatility smiles, demonstrating accuracy for exotic option pricing. See code at \href{https://github.com/rhesus1/Finance/tree/main/OptionPricing}{github.com/rhesus1/Finance/OptionPricing}.
